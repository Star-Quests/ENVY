# ENVY - Professional Crypto Trading Journal

![ENVY Logo](frontend/assets/icons/envy-emblem.svg)

## 🚀 Overview

ENVY is a professional-grade cryptocurrency trading journal and portfolio management platform designed for serious traders. Track every trade, analyze performance, and optimize your trading strategy with institutional-grade tools.

### ✨ Features

- **Live Portfolio Tracking** - Real-time P&L updates with millisecond precision
- **Independent Trade Timers** - Each trade runs its own timer for precise duration tracking
- **Mathematical Trade Planner** - Calculate perfect entries, exits, and trailing stops
- **Deep Customization** - Every aspect of the UI is customizable per user
- **Multi-Asset Support** - Track any cryptocurrency available on Bybit
- **Professional Analytics** - Win rate, profit factor, ROI, and risk/reward analysis
- **Admin Panel** - Full user management and platform configuration
- **Dark Theme by Default** - Luxury dark interface with light grey accents

## 📦 Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+ Modules)
- **Backend**: Node.js + Express
- **Database**: Supabase (PostgreSQL)
- **Real-time**: WebSocket for live price updates
- **Storage**: Cloudinary for image hosting
- **Hosting**: Render
- **Price Data**: Bybit API
- **Asset Logos**: CoinGecko API
